Project 14: Interview Preparation Chatbot with Real-Time Feedback
Problem Statement Students preparing for technical interviews lack personalized, real-time feedback on their
performance. Build an AI interview coach that conducts mock interviews (technical + behavioral), provides
real-time feedback on: answer quality, communication clarity, body language (via camera), filler word usage,
and adapts difficulty based on performance.
Expected Output
Video call interface with AI interviewer (avatar/voice)
Technical interview: DSA, system design, ML questions with follow-ups
Real-time feedback: filler word counter, speaking pace, eye contact score
Post-interview detailed scorecard: technical accuracy, communication, confidence
Personalized study plan based on identified weak areas
Progress tracking across multiple sessions
Company-specific interview prep (FAANG, startups, etc.)
Tech Stack: WebRTC for video, Whisper for transcription, MediaPipe for body language analysis, GPT-4o for
question generation and evaluation, React, TensorFlow.js, PostgreSQL

Methodology
1. Session Setup -- User selects interview type, role, difficulty. GPT-4o generates role-appropriate
questions.
2. Audio Pipeline (WebRTC + Whisper) -- Real-time audio streaming. VAD for speech segmentation.
Detect filler words via pattern matching on Whisper transcripts with word-level timestamps (WhisperX).
Track: WPM, pause duration, filler frequency.
3. Body Language (MediaPipe) -- MediaPipe Holistic (face mesh + pose + hands) per frame. Eye contact
via iris position relative to eye corners. Posture via shoulder alignment. Facial expressions via landmark
distances.
4. Answer Quality Assessment -- Whisper transcript + question sent to GPT-4o. Evaluate: relevance,
completeness, structure (STAR method), technical accuracy. Score 1-10 per dimension.
5. Real-Time Feedback Engine -- During: subtle on-screen cues ("Slow down", "Make eye contact"). After
each answer: brief assessment. After interview: comprehensive scorecard.
6. Progress Tracking -- Record sessions. Track improvement: filler word reduction, confidence trends,
answer quality.
